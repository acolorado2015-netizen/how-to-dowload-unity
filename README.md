- # 1️⃣ Descargar Unity Hub

1. Ir a la página oficial:

  👉 https://unity.com/download

2. Descargar **Unity Hub** para Windows.

3. Ejecutar el instalador y completar la instalación.

4. Abrir Unity Hub e iniciar sesión con cuenta Unity (Personal gratuita).

# 2️⃣ Instalar Unity Editor

1. En Unity Hub ir a:

  **Installs → Install Editor**

2. Seleccionar una versión **LTS (Long Term Support)**.

  Recomendado:

  - Unity 2022 LTS

  - Unity 2023 LTS

3. Hacer clic en **Install**.

# 3️⃣ Seleccionar Módulos Importantes

Durante la instalación marcar:

✔️ Microsoft Visual Studio Community

✔️ Windows Build Support (IL2CPP)

✔️ WebGL Build Support

✔️ Android Build Support (opcional)

Estos módulos permiten exportar proyectos a distintas plataformas.

# 4️⃣ Instalar Visual Studio Community

Si no se instaló automáticamente:

1. Ir a:

  👉 https://visualstudio.microsoft.com/es/vs/community/

2. Descargar **Visual Studio Community**.

3. Durante la instalación seleccionar:

✔️ Desarrollo de escritorio con .NET

✔️ Desarrollo de juegos con Unity

# 5️⃣ Configurar Unity con Visual Studio

1. Abrir Unity.

2. Ir a:

  **Edit → Preferences → External Tools**

3. En *External Script Editor* seleccionar:

  👉 Visual Studio

Esto permite editar scripts en C# correctamente.

# 6️⃣ Crear un Proyecto en Unity

1. Abrir Unity Hub.

2. Ir a **Projects → New Project**.

3. Seleccionar plantilla:

  - 3D Core (juegos 3D)

  - 2D Core (juegos 2D)

4. Asignar nombre al proyecto.

5. Seleccionar carpeta de guardado.

6. Hacer clic en **Create Project**.

# 7️⃣ Instalar Git

1. Ir a:

  👉 https://git-scm.com/download/win

2. Descargar Git para Windows.

3. Instalar con la configuración por defecto.

# 8️⃣ Instalar GitHub Desktop

1. Ir a:

  👉 https://desktop.github.com/

2. Descarga e instala.

3. Iniciar sesión con tu cuenta en GitHub.
